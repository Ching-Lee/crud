# crud
bootstrap+Ajax+SSM(maven搭建)实现增删改查
 ## 功能点：
- 分页
- 数据校验
- ajax
- Rest风格的URI：使用HTTP协议请求方式的动词，来表示对资源的操作（GET（查询），POST（新增），PUT（修改），DELETE（删除））
## 技术点
- 基础框架-ssm
- 数据库mysql
- 前端框架-bootstrap
- 项目依赖 -Maven
- 分页 -pagehelper
- 逆向工程-Mybatis Generator
